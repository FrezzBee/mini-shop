# Глосарій термінів Mini Shop

- **Product** — товар у каталозі
- **Order** — замовлення одного або кількох товарів
- **OrderItem** — один елемент замовлення (товар + кількість)
- **Payment** — платіж за замовлення
- **Catalog** — каталог товарів
- **Status** — стан замовлення або платежу (pending, paid, canceled)
